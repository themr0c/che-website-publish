Upgrading Che v7.41 on OpenShift

The workspace engine and authentication system used until Che version v7.41 are deprecated. Upgrading from version v7.41 requires manual steps.

Prerequisites

  • An instance of Che deployed on Supported platforms, using the default internal PostgreSQL database, and with OpenShift OAuth enabled. See Configuring OpenShift OAuth.

  • Command line tools oc, curl, and jq are available.

  • The host running the upgrade commands is running on Linux.

Procedure

  1. Optionally push back changes to Git repositories in all workspaces, stop all workspaces to avoid user experience degradation, and backup Che data. See Backup and recovery.

  2. Set the environment variables to use in the migration scripts:

    export INSTALLATION_NAMESPACE=eclipse-che (1)
export OPERATOR_NAMESPACE=eclipse-che     (1)
export PRODUCT_ID=eclipse-che
export PRODUCT_DEPLOYMENT_NAME=che
export PRODUCT_OPERATOR_NAME=che-operator
export PRODUCT_OLM_STABLE_CHANNEL=stable
export PRODUCT_OLM_CATALOG_SOURCE=community-operators
export PRODUCT_OLM_PACKAGE=eclipse-che
export PRE_MIGRATION_PRODUCT_SHORT_ID=che
export PRE_MIGRATION_PRODUCT_DEPLOYMENT_NAME=che
export PRE_MIGRATION_PRODUCT_OPERATOR_NAME=che-operator
export PRE_MIGRATION_PRODUCT_CHE_CLUSTER_CR_NAME=eclipse-che
export PRE_MIGRATION_PRODUCT_IDENTITY_PROVIDER_DEPLOYMENT_NAME=keycloak
export PRE_MIGRATION_PRODUCT_SUBSCRIPTION_NAME=eclipse-che
    1 Adapt to the namespace where Che is installed.

  3. Download and execute the migration scripts: 1-prepare.sh, 2-migrate.sh, 3-subscribe.sh, and 4-wait.sh.

    chmod +x ./1-prepare.sh ./2-migrate.sh ./3-subscribe.sh ./4-wait.sh
./1-prepare.sh (1)
./2-migrate.sh (2)
./3-subscribe.sh (3)
./4-wait.sh (4)
    1 Shuts down Che and Keycloak, fetch users data and dumps Che database.
    2 Fetches Che Keycloak and database data, and repopulates the database with updated data.
    3 Deletes Che Operator and Keycloak resources, updates CheCluster CR and creates a new Che Operator subscription.
    4 Waits until Che is ready (it can take more than 5 minutes).
Verification

  • Open Che dashboard and log in.

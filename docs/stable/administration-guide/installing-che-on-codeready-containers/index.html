Installing Che on CodeReady Containers

This chapter describes the creation of a single-node OpenShift cluster for deploying a Che instance using CodeReady Containers. To install Che, use the`chectl` command-line tool or the OpenShift web console GUI.

Using chectl to install Che on CodeReady Containers

This section describes how to install Che on CodeReady Containers using chectl.

Prerequisites

  • The chectl management tool is available. See Using the chectl management tool.

  • An installation of CodeReady Containers. See Installing CodeReady Containers.

  • The path to the user’s CodeReady Containers pull secret is known as <pull-secret-file>.

  • Available memory: 12 GB or more.

  • Available disk size: 64 GB or more.

Procedure

  1. Remove any previous cluster:

    $ crc delete

  2. Configure your host machine for CodeReady Containers:

    $ crc setup

  3. Start the CodeReady Containers virtual machine with at least 12 GB of memory and 64 GB of disk space. Also, specify the path to your pull secret and take note of the password for the kubeadmin user displayed at the end of the CodeReady Containers initiation:

    $ crc start --memory 12288 --disk-size=64 --pull-secret-file <pull-secret-file>

  4. Enable access to the oc command line interface embedded in CodeReady Containers:

    $ eval $(crc oc-env)

  5. Install the Che instance using chectl:

    $ chectl server:deploy --platform crc

  • Product documentation for Red Hat CodeReady Containers.

  • CodeReady Containers on GitHub.

Using the OpenShift web console to install Che on CodeReady Containers

Prerequisites

  • An installation of CodeReady Containers. See Installing CodeReady Containers.

Procedure

  1. To configure your host machine for CodeReady Containers, run:

    $ crc setup

  2. To remove any previous cluster, run:

    $ crc delete

  3. To start the CodeReady Containers virtual machine with at least 12 GB of RAM, run:

    $ crc start --memory 12288

  4. When prompted, supply your user pull secret.

  5. Take note of the password for the user kubeadmin, which the crc start --memory 12288 command outputs.

  6. To access the OpenShift web console, run:

    $ crc console

  7. In the web console, log in with the username developer and the password developer.

  8. In the top right corner of the OpenShift web console, click developer → Log out.

  9. Log in with the kubeadmin username and the corresponding password.

  10. Follow the procedure for Installing Che on OpenShift 4 using OperatorHub.
